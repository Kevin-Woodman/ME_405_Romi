Detailed Description

MotorEncoder is the low-level control loop for Romi's motors.

This class contains an initialization function and a generator function to be used as a task. Each instance of this class represents either the left or right motor encoder pair.

Defines the task for MotorEncoder.

This generator function defines the task for the MotorEncoder, is starts in an Initialization state before alternating between sensing and controlling states. It uses a PID to control the motor to a desired angular velocity.
